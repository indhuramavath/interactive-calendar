# Interactive Calendar Component

This project implements an interactive calendar where users can select a date range and attach notes.

## Features

- Select a start date and end date
- Highlight the selected date range
- Add notes for the selected range
- Notes are saved using localStorage
- Interactive UI built with React

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## How to Run the Project

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the application in the browser:

```
http://localhost:5173
```

## Functionality

1. Click a start date
2. Click an end date
3. The date range gets highlighted
4. Write a note for that range
5. Save the note
6. Notes remain saved using localStorage
